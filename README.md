## board-project
사용자가 글을 작성하고, 다른 사용자와 공유할 수 있는 게시판 웹 애플리케이션입니다.

***

### 기술 스택
+ Node.js
+ Express.js
+ MongoDB
+ EJS
+ HTML / CSS

***

### 기능
1. 로그인 / 회원가입
2. 게시글 작성 / 수정 / 삭제
3. 게시글 조회
4. 댓글 작성 / 삭제
5. 게시글 검색

***

### 실행
```
git clone https://github.com/ahyng/board-project.git
```

``` 
npm run start 
```
